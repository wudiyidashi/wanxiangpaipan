# PRD: 六爻卦名起卦方式

## Goal
新增「卦名卦」起卦方式：用户直接给定月建（如巳月）、日干支（如乙亥日），选择本卦（如水天需）与可选变卦（如泽风大过），即完成排盘。用于录入古籍卦例或他处得到的卦。

## Requirements
- R1 CastMethod 新增 guaName（卦名卦）；LiuYaoSystem.supportedMethods 加入
- R2 动爻反推：本卦与变卦逐位比较，阴阳不同之爻为动爻（阳→9 阴→6），相同为静（7/8）；无变卦=全静；变卦=本卦视为全静
- R3 月日自定：LunarInfo 的 yueJian/riGan/riZhi/riGanZhi/kongWang 按用户输入构造（空亡由日干支推）；monthGanZhi 置为月建支保证显示一致；年干支沿用起卦当时（仅影响太岁入爻）；时辰置空
- R4 UI：起卦方式选「卦名卦」时显示 月建下拉(12支)、日干支下拉(60甲子)、本卦下拉(64卦名)、变卦下拉(无变卦+64)
- R5 校验：卦 id 合法、月建为地支、日干支为六十甲子、变卦可为空

## Acceptance Criteria
- AC1 巳月乙亥日 需→大过 卦例：初爻四爻动，空亡申酉，月建巳（单测）
- AC2 无变卦=全静；变卦同本卦=全静
- AC3 UI 可完成整个流程并进入结果页（实机）
- AC4 全量测试保持绿；其他系统起卦不受影响

## Out of Scope
- 年干支/时辰自定；其他系统的卦名录入
